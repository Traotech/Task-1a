# Advanced Todo Card (HNG Stage 1)

This project is an advanced Todo Card component built as part of my HNG Internship Stage 1 task. It extends the basic Todo card into a more interactive and stateful component using HTML, CSS, and JavaScript.



##  Features

-  Edit Mode (update title, description, priority, and due date)
-  Status Control (Pending, In Progress, Done)
-  Checkbox & status synchronization
-  Priority Indicator (Low, Medium, High with visual styling)
-  Expand / Collapse description
-  Dynamic Time Handling (days, hours, minutes)
-  Overdue detection with visual feedback
-  Accessibility support (labels, aria attributes, keyboard navigation)
-  Fully responsive design



##  What Changed from Stage 0

- Added editable task functionality
- Introduced status control with synchronization logic
- Improved time logic with granular updates and overdue detection
- Added priority visual indicators
- Implemented expand/collapse for long descriptions
- Enhanced accessibility and keyboard usability

---

## Technologies Used

- HTML5 (Semantic structure)
- CSS3 (Flexbox, responsive design)
- Vanilla JavaScript (DOM manipulation, state management)
